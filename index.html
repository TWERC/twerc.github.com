TWERC

Tech Writers, Editors and Reporters in China.

class TWERC {
    if ("can code" || "can hack" || "can play around with IT devices") {
        if ("想自由出入各类技术大会" || "想写文" || "想出书" || "想采访大牛") {
            if (("不知道如何入门" || "不知道如何提升") && ("写过一些东西")) {
                do {"参加TWERC训练营学习";}
            } elseif ("有写作心得") {
                do {"在TWERC训练营分享";}
            }
        } elseif ("是编辑" || "是技术社区经理" || "是技术布道师") {
            if ("缺社区作者" || "缺社区编辑" || "缺社区记者") {
                do {"来TWERC训练营培养壮丁";}

什么是TWERC？

  • Tech Writers, Editors and Reporters in China.
  • 中文界技术作者、编辑与记者。

什么是TWERC训练营？

  • 以培养TWERC为目的的线下聚会活动。
  • 训练内容包括但不限于
    • 国内各个技术媒体的背景与稿件需求介绍
    • 如何进行高质量的写作
    • 如何进行事件报道
    • 如何进行事件评论
    • 如何做采访
    • 如何推广
  • 分享嘉宾包括但不限于
    • 国内外优秀的技术媒体的总编、编辑们
    • 国内外优秀的技术博主
    • 国内外优秀的作者、译者们
    • 国内外优秀的技术讲师、技术社区经理、布道师们

为什么要发起TWERC？

2013年3月的某一个傍晚，InfoQ中文站的编辑们在夕阳下一边走向羽毛球场，一边讨论技术社区的线索很多、但拉不到足够的壮丁来进行报道的问题。

“主动培养一些社区作者，如何？”

大家都觉得这是一件很有价值的事，何乐而不为？

培养原创作者群体事关整个行业，所以，大家一起来做这件事情，价值会更大。

谁在组织TWERC？

  • InfoQ中文站
  • 欢迎更多志同道合的媒体、出版社、关注技术推广的朋友们加入（见下面的“如何加入”部分）

听起来很有意思！我想一起来组织TWERC，要如何加入？

  • 联系InfoQ中文站CEO霍泰稳，或其他已知的TWERC组织者（见上一部分）。
  • 或者，您也可以直接申请加入https://groups.google.com/d/forum/twerc
  • 您可以以如下方式参与TWERC的组织工作：
    • 分享一个话题，由您自己分享或推荐人选分享均可
    • 赞助活动场地（可以是公司的办公室）
    • 帮助我们宣传TWERC
    • 在您所在的城市组织当地的TWERC活动

我想来TWERC训练营学习，如何申请？

  • 申请加入https://groups.google.com/d/forum/twerc，以关注里面发布的消息
  • 成功加入群组后，可以通过这份Google文档查看下次活动的相关信息和报名地址。
